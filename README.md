# LimonCELLo — DT neuron & cilia analysis

![alt text](image.png)

Overview
--------
LimonCELLo contains analysis code for processing 3D Imaris (.ims) microscopy
datasets, focusing on associating cilia to neurites and classifying their location (neurite or soma). 

Check out the [tutorial notebook](tutorial/tutorial.ipynb)

1. Install runtime dependencies:

```powershell
pip install numpy scipy pandas matplotlib pyclesperanto_prototype scikit-image
```
